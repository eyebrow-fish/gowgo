go with go - If Statements

If statements are a crucial part to any program.

If statements start with the if keyword. Directly after is the condition to be evaluated. Then the block to executed if the condition is true.

Sometimes our conditions may be false and we want to do something else in this case. We do this with else or else if. The difference being that else can only occur once and has no condition.

Scoping variables are also an important feature. These are variables which are only accessible in the if conditions and blocks.

package main

import (
    "fmt"
)

func main() {
    x := true
    if x {
        fmt.Println("x is true")
    } else {
        fmt.Println("x is false")
    }
    // scoped variables
    if good := "dog"; good == "dog" {
        // good can be used here
        fmt.Printf("good %s\n", good)
    } else {
        // good can still be used
        fmt.Printf("not good %s\n", good)
    }
    // good can no longer be used
    if num := 50; num < 5 {
        fmt.Println("tiny")
        // scoped variables can be used in
        // "else if" conditions
    } else if num < 20 {
        fmt.Println("medium")
    } else if num < 50 {
        fmt.Println("large")
    } else {
        fmt.Printf("%d is big\n", num)
    }
}
